# AICoding

该工程试图让个人编程方法，使用AI Agent来复刻。

很多人以为AI编程的核心问题在于LLM，以及代码索引技术。 这当然是非常好的研究方向。但个人以为代码本身的组织也至关重要。
这就像社会治理一样，即便再聪明的人也难以解决系统性崩塌问题。

所以问题的关键在于约束，即如何让代码始终保持清晰整洁，语义友好的架构。


# 核心思想

1. 在某个关注面下，任何事物在系统中都有且仅有一个最合适的逻辑位置。
2. 严格约束模块之间的依赖关系，使得增量需求的描述非常清晰。
3. 表达潜在约束，这点可以基于Rust进行改进。

这些手段不是通过提示词来约束，而是通过工具（含编译工具）来实现。
